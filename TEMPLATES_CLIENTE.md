```

Olá [NOME DO CLIENTE],

Preparei este guia com soluções para problemas comuns:

═══════════════════════════════════════════════════════════════
🔧 PROBLEMAS COMUNS E SOLUÇÕES
═══════════════════════════════════════════════════════════════

PROBLEMA: "Licença Inválida" ao acessar
✅ SOLUÇÃO: 
   - Verifique se está usando o link correto
   - Link correto: https://app.seudominio.com.br
   - Não use: http (sem 's'), IP direto, ou outro domínio

───────────────────────────────────────────────────────────────

PROBLEMA: Dados sumiram / Tela em branco
✅ SOLUÇÃO:
   1. Restaure o último backup (botão "Restaurar Backup")
   2. Se não tem backup, os dados podem estar perdidos
   3. SEMPRE faça backup semanal!

───────────────────────────────────────────────────────────────

PROBLEMA: Sistema está lento
✅ SOLUÇÃO:
   1. Feche outras abas do navegador
   2. Limpe o histórico (mas NÃO limpe dados do site!)
   3. Atualize o navegador para última versão
   4. Reinicie o computador

───────────────────────────────────────────────────────────────

PROBLEMA: Não consigo imprimir
✅ SOLUÇÃO:
   1. Use Google Chrome (melhor para impressão)
   2. Ctrl + P para abrir impressão
   3. Configure como "Paisagem"
   4. Ou salve como PDF

───────────────────────────────────────────────────────────────

PROBLEMA: Valores não calculam corretamente
✅ SOLUÇÃO:
   1. Use números sem pontos: 1500 (não 1.500)
   2. Use vírgula para decimais: 10,50
```
